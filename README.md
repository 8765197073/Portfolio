# Personal Portfolio

## 🚀 Overview
My professional portfolio website showcasing my journey as a Business Analyst and developer. This site highlights my core projects, professional experience, and technical skill set.

## 🛠 Tech Stack
* **Framework:** React / Next.js
* **Styling:** Tailwind CSS / CSS Modules
* **Deployment:** Vercel

## 🔑 Key Features
* **Project Showcase:** Interactive display of my professional and technical work.
* **Responsive Design:** Optimized for seamless viewing on mobile, tablet, and desktop.
* **Fast Performance:** Server-side rendering for optimal load speeds.
